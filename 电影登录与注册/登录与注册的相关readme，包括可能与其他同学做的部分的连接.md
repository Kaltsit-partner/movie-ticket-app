# 猫猫电影网站的登录与注册
## 队友请读：
1.`login.html` - 登录页、`register.html`- 注册页、`forgot-password.html` - 忘记密码页：都包含了一个跳转首页`index.html`，为了确认功能使用我简单制作了一个页面，需要替换到制作完成的`index.html`
2.因为我设置了一个logo部分跳转首页，为了规避版权问题，ai了一个logo图标，并将“猫眼电影”改名“猫猫电影”的logo，如果与其他同学的设计相矛盾需要替换，麻烦替换一下`login.html` - 登录页、`register.html`- 注册页、`forgot-password.html` - 忘记密码页：的logo图标的链接
3.我将auth.js,style.css,utils.js、两个图片和html放在同一文件夹中，如果之后同学按文件类型放文件夹，麻烦改一下html代码中跟auth.js,style.css,utils.js、两个图片相关的链接
 


## 功能特点
- ✅ 每个页面的logo部分都可以点击跳转首页`index.html`
- ✅ 用户注册：使用邮箱进行注册，要求密码包含字母和数字，且大于8位
- ✅ 用户登录：用户登录并包含下端“忘记密码？”和“注册用户”跳转链接
- ✅ 忘记密码：使用邮箱进行密码重置

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage (模拟数据库)



## 页面说明

- `index.html` - 首页*需要换正确的链接
- `login.html` - 登录页
- `register.html` - 注册页
- `forgot-password.html` - 忘记密码页
